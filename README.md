## 📦 Реализованные проекты

### 01 · Бот для онлайн-курса
Полноценная образовательная платформа внутри Telegram.

- Оплата через вебхук — доступ только после покупки
- 4 формата контента: язык (RU/UA) × музыка (вкл/выкл)
- Отслеживание прогресса, вопросы после каждого урока
- Адаптивные напоминания по текущему уроку
- История ответов, сброс прогресса

**Стек:** `Python` `aiogram 3` `PostgreSQL` `Railway` `WayForPay` `Resend`

### 02 · Бот для записи на туры — [@nhatrangobot](https://t.me/nhatrangobot)
Автоматизация записи клиентов на групповые туры.

- Самозапись клиентов — выбор тура, даты, ввод ФИО
- Админ-панель: подтверждения, оплаты, управление датами
- Автонапоминания за 24 часа до тура
- Запрос отзыва после тура
- Экспорт списка группы в CSV

**Стек:** `Python` `aiogram 3` `aiosqlite` `APScheduler` `Railway` `GitHub`
