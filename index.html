Проект 02
Туризм / сервисный бизнес

Бот для записи на туры — @nhatrangobot

Автоматизация записи клиентов на групповые туры. Заменяет менеджера по записи: клиенты бронируют сами, получают напоминания и оставляют отзывы.

Самозапись клиентов — выбор тура, даты, ввод ФИО Подтверждение записей и управление оплатами в админке Автонапоминания за 24 часа до тура Запрос отзыва после завершения тура Управление турами: даты, места, цены, фото/видео Экспорт списка группы в CSV, рассылка всем клиентам
Python aiogram 3 aiosqlite APScheduler Railway GitHub
